A [Flarum](http://flarum.org) extension. ct

## Installation

Install with composer:

```sh
composer require huoxin/custom-time:"*"
```

## Updating

```sh
composer update huoxin/custom-time:"*"
php flarum migrate
php flarum cache:clear
```
